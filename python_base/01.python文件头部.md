# Python文件头部注释的含义

## 1、指定解释器

window系统：
```python
#!/usr/bin/python
```

linux系统：

```python
#!/usr/bin/env python
```



## 2、指定.py文件的编码类型

```python
# -*- coding: utf-8 -*-
```

或者

```python
# coding: utf-8
```



## 自定义的文件头部

```python
#!/usr/bin/env python

# -*- encoding: utf-8 -*-

'''
@Author  :   {YourName}

@License :   (C) Copyright 2013-2017, {YourCompany}

@Contact :   {YourEmail}

@Software:   ${PRODUCT_NAME}

@File    :   ${NAME}.py

@Time    :   ${DATE} ${TIME}

@Desc    :

'''

import...


if __name__=='__main__':
    pass

```
